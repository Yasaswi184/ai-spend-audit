# Metrics

## Product Metrics

### Activation Metrics
- Number of completed audits
- Audit completion rate
- Time to first audit

### Engagement Metrics
- Returning users
- Average session duration
- Audits per user

### Conversion Metrics
- Free to paid conversion rate
- CTA click-through rate
- Waitlist signups

### Revenue Metrics
- Monthly recurring revenue (MRR)
- Average revenue per user (ARPU)
- Customer lifetime value (LTV)

### Retention Metrics
- Weekly active users
- Churn rate
- Repeat audit frequency

---

## Technical Metrics

- Lighthouse performance score
- Deployment uptime
- API response time
- Error rate
