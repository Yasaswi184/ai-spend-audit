# AuditAI

AuditAI is an AI spend optimization platform that helps startups identify wasted spending on AI tools like ChatGPT, Claude, Cursor, and GitHub Copilot.

Users can input their current AI stack, monthly spend, and team size to receive instant recommendations, cheaper alternatives, and estimated annual savings.

## Features

- AI spend audit engine
- Savings recommendations
- Alternative tool suggestions
- Responsive landing page
- Interactive audit form
- Real-time results
- Vercel deployment

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Vercel
