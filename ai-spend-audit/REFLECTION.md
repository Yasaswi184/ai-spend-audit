# Reflection

## 1. Hardest Bug

The hardest issue was fixing the component import paths in Next.js. Initially the Navbar component was not resolving correctly because the folder structure was incorrect. I debugged by checking aliases, folder nesting, and import syntax. Moving the components into the correct app/components folder resolved the issue.

## 2. Reversed Decision

Initially I wanted to build a much more complex dashboard with authentication and backend storage. Midway through the project I realized that shipping a polished MVP was more important than adding unfinished features. I simplified the scope and focused on usability and deployment quality.

## 3. Week 2 Improvements

If given another week, I would add:
- Real AI API integrations
- Persistent database storage
- User authentication
- PDF export
- Better analytics dashboard
- Advanced savings recommendations

## 4. AI Tool Usage

I used ChatGPT for debugging, architecture planning, and UI suggestions. I used AI carefully and verified outputs manually. One example where AI was wrong was a broken import structure suggestion which caused module resolution errors. I fixed it by restructuring the components directory manually.
